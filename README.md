# 文芸同人誌「北斗」ホームページ イメージ案

## ファイル構成

```
hokuto_website/
├── index.html        ← トップページ（最新号表紙＋目次、右サイドバー付き）
├── backnumber.html   ← バックナンバーページ
├── blog.html         ← ブログ・お知らせページ
├── about.html        ← 北斗についてページ
├── images/           ← 表紙画像などを置くフォルダ（作成してください）
└── README.md
```

## ブラウザで確認する方法

`index.html` をダブルクリックするとブラウザで開けます。

---

## 各ページの更新方法（管理者向け）

### トップページ（index.html）の更新

新しい号が出たら：
1. `index.html` をメモ帳（またはテキストエディタ）で開く
2. 「最新号のご案内」セクション内の号数・日付・目次を書き換える
3. 表紙画像を `images/` フォルダに入れ、`src="images/cover_○○○.jpg"` と指定する

### バックナンバーページ（backnumber.html）の更新

1. 最新号のカード（`<div class="bn-card">`）をコピーして新しい号を追加する
2. 古い号は下に追加していく

### ブログページ（blog.html）の更新

1. 新しい記事（`<article class="blog-post">`）をコピーして先頭に追加する
2. 日付・タグ・タイトル・本文を書き換える

## 今後のプラットフォーム候補（検討用）

| サービス | 特徴 | 高齢者向けしやすさ |
|---------|------|-----------------|
| **WordPress.com** | GUIで記事投稿、プラグイン豊富 | ★★★★ |
| **はてなブログ** | 日本語、シンプルUI | ★★★★★ |
| **Wix** | ドラッグ＆ドロップ、日本語 | ★★★★ |
| **さくらブログ** | 国内サービス、サポート良好 | ★★★★ |
| **静的HTML（現状）** | 低コスト、軽量 | ★★（要テキスト編集スキル） |

※ 高齢者でも更新しやすいのは「はてなブログ」「WordPress.com」が候補としてあがりやすいです。

## メモ

- 現在の表紙画像（HEIC形式）はそのままではウェブで使えないため、JPGまたはPNGに変換が必要です
- 色テーマ：濃紺（#1a1a2e）× 深紅（#8b1a1a）× ベージュ（#f7f4ef）＝格調ある文芸誌らしいデザイン
